# Reflection

During this project, I primarily used GitHub Copilot and ChatGPT as AI development assistants. I used GitHub Copilot inside Visual Studio Code to generate code, implement backend and frontend features, write tests, and update project documentation. I used ChatGPT mainly to plan the work, create structured prompts for Copilot, review the implementation workflow, and ensure that every step followed the assignment requirements. Using both tools together helped me keep the implementation organized while maintaining control over the final result.

One moment where AI was particularly helpful was during the implementation of the Due Dates feature. By breaking the work into small prompts, Copilot generated the models, storage updates, API routes, frontend changes, and tests incrementally instead of attempting to modify the entire project at once. This made it easier to inspect each change, verify it, and continue with confidence.

One moment where AI slowed me down occurred during the frontend implementation of the Task Comments feature. Although Copilot generated the required code, the comments section did not appear in the edit modal because the integration with the existing modal lifecycle was incomplete. I had to inspect the generated code, identify that the feature was not connected correctly, and provide a more focused prompt so that only the integration issue was fixed. This reinforced the importance of reviewing AI-generated code instead of assuming it is correct.

A place where my own review changed the final result was during the Break Test and validation of the comment feature. After intentionally introducing the validation bug, I restored the correct implementation and noticed that the validator still accepted blank and whitespace-only comments. By reviewing the failing tests and correcting the validator, I ensured that comments were trimmed correctly and invalid input returned HTTP 422 as required. This demonstrated that human review and testing remain essential, even when AI generates most of the implementation.
